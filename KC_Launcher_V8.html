KC Launcher 8.0 เวอร์ชันนี้มีอะไรบ้าง?

-ระบบเช็ต Path FiveM และ TeamSpeak 3
-ปรับปรุงการเขียนโค้ดใหม่ทั้งหมด

-เชื่อมต่อ Server FiveM อัตโนมัติ (รองรับ IP,JoinID,Http)
-ผู้ใช้สามารถดูและคัดลอก Steam ID และ Username ผ่านปุ่มการตั้งค่า
