วิธีตั้งค่า KC Launcher V7.1

[หมวด FiveM]

*หัวข้อ* หน้าที่ของแต่ละสี
สีชมพูคือเลือกโหมดเชื่อมต่อที่จะใช้ใน Launcher โดยที่จะนำตัวเลข 0,1,2 มาใส่เป็นตัวเลขแทนค่าโหมดต่างๆ เช่น
สีแดงคือโหมดเชื่อมต่อแบบ IP แทนค่าเป็นเลข 0
สีส้มคือโหมดเชื่อมต่อแบบ Join ID แทนค่าเป็นเลข 1
สีเหลืองคือโหมดเชื่อมต่อแบบ Http แทนค่าเป็นเลข 2

*หัวข้อ* ตั้งค่าโหมดเชื่อมต่อ
โหมดเชื่อมต่อแบบ IP จะต้องกำหนดค่าให้ FiveM_IP และ FiveM_Port แบบตัวอย่างที่ให้ไป
โหมดเชื่อมต่อแบบ Join ID จะต้องกำหนดค่าให้ FiveM_JoinID แบบตัวอย่างที่ให้ไป
(คุณสามารถดูผ่าน https://servers.fivem.net/ และค้นหาเซิร์ฟเวอร์ของคุณ Join ID คุณจะอยู่ไกล้ๆกับปุ่ม Connect)
โหมดเชื่อมต่อแบบ Http จะต้องกำหนดค่าให้ FiveM_Http ซึ่งจะเป็นในรูปแบบโดเมนเนม หรือ IP ที่นำหน้าด้วย http://127.0.0.1:30120/ แบบตัวอย่างที่ให้ไป

*หัวข้อ* วิธีเลือกโหมดเชื่อมต่อ
ให้กำหนดเลขแทนค่าให้ FiveM_Connect_Mode เป็น โหมดที่คุณจะใช้งาน เช่น 0,1,2 โปรดเลือกโหมดใดโหมดหนึ่ง แบบตัวอย่างที่ให้ไป
(เลขแทนค่าของแต่ละโหมดจะอยู่ใน *หัวข้อ* หน้าที่ของแต่ละสี )
