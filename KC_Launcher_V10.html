KC Launcher 9.0 เวอร์ชันนี้มีอะไรบ้าง?

-ระบบอัพเดท Launcher อัตโนมัติ
-ระบบอัพเดทแพ็คเกจ Fujino.KCLauncher.Connect อัตโนมัติ
-รองรับ Background ที่เป็นไฟล์ MP4 เท่านั้น
-เล่นไฟล์ MP4 แบบวนซ้ำ
-เพิ่ม UI 'ประกาศและกิจกรรม' และ 'ประกาศจากเซิร์ฟเวอร์'
-เข้ารหัสไฟล์ตั้งค่า
-ตั้งค่าทั้งหมดจากภายนอกผ่านโปรแกรม KC Launcher V10 - Settings
-ป้องกันการเปิดโปรแกรมซ้ำกัน
-เชื่อมต่อ Server FiveM อัตโนมัติ (รองรับ IP,JoinID)
-ผู้ใช้สามารถดูโปรไฟล์ Steam และคัดลอก Steam ID
-ระบบแสดงสถานะเซิฟเวอร์และจำนวนผู้เล่น

ประวัติการอัปเดตและแก้ไข
