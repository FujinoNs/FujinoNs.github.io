KC Launcher 7.0 เวอร์ชันนี้มีอะไรบ้าง?

-เพิ่มระบบเช็ต Path FiveM และ TeamSpeak 3
-ปรับปรุงการเขียนโค้ดใหม่ทั้งหมด สามารถแก้ไขipและลิ้งค์ต่างๆได้ในไฟล์เดียว
-เพิ่มระบบเช็คเวอร์ชั่น Launcher
-เชื่อมต่อ Server TS3 อัตโนมัติ
-เชื่อมต่อ Server FiveM อัตโนมัติ (ไม่รองรับโดเมนเนม)
-ปรับปรุงระบบ Uninstall ให้ใช้งานได้ง่ายเพียง2คลิ๊ก ไม่จำเป็นต้องเข้าไปกด Uninstall ใน Programs and Features อีกต่อไป
-ผู้ใช้สามารถดูและคัดลอก Steam ID และ Username ผ่านปุ่มตั้งค่า
-เพิ่มระบบติดตั้งปลั๊กอิน TokoVoip ให้ง่ายขึ้น ไม่จำเป็นต้องเข้าไปโหลดในเว็บเหมือนเวอร์ชั่นก่อนหน้า
-เพิ่มตัวเล่นวีดีโอ(YouTube)ภายใน Launcher สำหรับปุ่ม "การตั้งค่า Plugin"

ประวัติการอัปเดตและแก้ไข

-ใช้งานแพ็คเกจ: Fujino.KCLauncher.Connect

-เพิ่มโหมดการเชื่อมต่อโดยที่ผู้เล่นไม่จำเป็นต้องตั้งค่า Path FiveM และ TeamSpeak

เวอร์ชั่น 7.2
